# PyBaMM Simulations — Electrochemical Battery Models

Exploratory Python scripts for electrochemical battery modelling using the PyBaMM (Python Battery Mathematical Modelling) framework.

## Contents

- **First_Simulation** — Initial PyBaMM setup and basic simulation runs to validate the environment and workflow
- **P2D(DNF)_Model** — Work-in-progress implementation of the Doyle-Fuller-Newman (DFN) / P2D electrochemical model

## Tech stack

- Python
- PyBaMM

## Context

These scripts were used to explore Python-based alternatives to ANSYS for battery electrochemical modelling, as part of R&D work on printed battery components.
